# Development Archive

This folder contains test scripts, debugging tools, and backtest iterations created during the development process.

## Contents

### Backtest Scripts
- `aggressive_strategy_backtest.py` - High leverage testing
- `backtest_multi_timeframe_gp.py` - Multi-timeframe Fibonacci analysis
- `fib_with_coinglass_backtest.py` - Sentiment-enhanced backtesting
- `run_golden_pocket_backtest.py` - Core golden pocket backtests

### Testing & Verification
- `test_*.py` - Various component and integration tests
- `check_*.py` - Position and data verification scripts
- `debug_*.py` - Debugging tools for golden pocket detection
- `verify_*.py` - Strategy validation scripts

### Logs
- `*.log` - Shadow mode testing logs and output

## Purpose

These files demonstrate the thorough testing and iteration process used to develop and validate the trading strategy. While not part of the production system, they provide transparency into the development methodology.

See `BACKTEST_LOG.md` in the root directory for detailed test results and findings.
